IoT?

The internet of things (IoT) is the network of physical devices, vehicles, buildings and other items—embedded with electronics, software, sensors, actuators, and network connectivity that enable these objects to collect and exchange data
(wikipedia)

Wemos d1 mini

Wemos d1 mini

  • wifi board based on ESP-8266EX
  • 11 digital input/output pins, all pins have interrupt/pwm/I2C/one-wire supported
  • A Micro USB connection
  • Compatible with Arduino
  • Shields!

DHT22

  • Temperature: -40~80°C (±0.5°C)
  • Humidity: 0-99.9%RH (±2%RH)
  • To get real-time data, you need read twice (measure after read)

HTTP LED

  • Switch onboard LED via http

Send temperature

  • No server, send values as HTTP get
  • Longer sleep cycles, less energy consumption
